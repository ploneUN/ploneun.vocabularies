ploneun.vocabularies Installation
---------------------------------

To install ploneun.vocabularies using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``ploneun.vocabularies`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        ploneun.vocabularies

* Re-run buildout, e.g. with:

    $ ./bin/buildout
